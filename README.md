## What it is

HEIRLOCK is a non-custodial financial operating system for the SoSoValue stack:

- **SoSoValue Terminal** — research, ETF, macro, news  
- **SSI** — index analytics + official Base allocate/earn via [ssi.sosovalue.com](https://ssi.sosovalue.com/)  
- **SoDEX** — per-user EIP-712 trading relay  
- **ValueChain** — WealthPolicy, continuity modes, ActionLog  

## Product flow

1. Connect wallet (SIWE)  
2. Enable Skills (Family Office on by default)  
3. Living Loop: Terminal evidence → proposal → risk preflight  
4. Allocate on SSI app when minting/staking; trade proxies on SoDEX  
5. Track fills and outcomes; Continuity modes Alive → Guardian → Heir  

## Stack

Fastify · Prisma/Supabase · TanStack Start · wagmi/viem · ValueChain contracts · SoDEX EIP-712 relay
